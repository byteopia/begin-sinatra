Begin here.
==========
All the things you need to start a *Sinatra* project you'll never finish.

Clone
----
```
$ git clone git@github.com:byteopia/begin-sinatra.git && cd begin-sinatra
```

Install
-------
```
$ bundle
$ npm install
$ npm install -g bower
$ bower install
```

Gulp
-----
```
$ gulp serve
```

Ruby
-----
```
$ ruby app/app.rb
```
